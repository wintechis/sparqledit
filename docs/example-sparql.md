## exmaple query 1

* basic BGP
* blank node construct (n-ary relation)

```
PREFIX foaf: <http://xmlns.com/foaf/0.1/>
PREFIX schema: <http://schema.org/>
SELECT ?name ?birthdate ?weight
WHERE {
  ?patient a schema:Patient ;
    foaf:name ?name ;
    schema:birthDate ?birthdate .
  ?patient schema:weight [
    schema:value ?weight
  ]
}
```

autogenerated update query when changing the patient's name:
```
PREFIX foaf: <http://xmlns.com/foaf/0.1/>
PREFIX schema: <http://schema.org/>
DELETE { <http://example.org/patient/1> foaf:name "John Doe". }
INSERT { <http://example.org/patient/1> foaf:name "John Doe 2". }
WHERE {
  <http://example.org/patient/1> <http://www.w3.org/1999/02/22-rdf-syntax-ns#type> schema:Patient;
    foaf:name "John Doe";
    schema:birthDate "1990-10-23"^^<http://www.w3.org/2001/XMLSchema#date>;
    schema:weight ?g_30.
  ?g_30 schema:value "80.5"^^<http://www.w3.org/2001/XMLSchema#decimal>.
}
```

autogenerated update query when changing the patient's weight:

## example query 2

* basic BGP
* blank node construct (n-ary relation)
* optional triple

```
PREFIX foaf: <http://xmlns.com/foaf/0.1/>
PREFIX schema: <http://schema.org/>
SELECT * #?name ?family ?birthdate ?weight
WHERE {
  ?patient a schema:Patient ;
    foaf:name ?name .
  OPTIONAL {  
    ?patient foaf:familyName ?family . 
  }
  OPTIONAL {  
    ?patient schema:birthDate ?birthdate .
  }
  ?patient schema:weight [
    schema:value ?weight
  ] .
  OPTIONAL {
    ?patient schema:memberOf ?medoffice .
    ?medoffice schema:address [
      schema:streetAddress ?medoffice_street ;
      schema:postalCode ?medoffice_zip  
    ] .
  }
}
```
autogenerated update query when changing patient 123's weight:
```
PREFIX foaf: <http://xmlns.com/foaf/0.1/>
PREFIX schema: <http://schema.org/>
DELETE { ?g_118 schema:value "125"^^<http://www.w3.org/2001/XMLSchema#decimal>. }
INSERT { ?g_118 schema:value "126"^^<http://www.w3.org/2001/XMLSchema#decimal>. }
WHERE {
  <http://example.org/patient/123> <http://www.w3.org/1999/02/22-rdf-syntax-ns#type> schema:Patient;
    foaf:name "Max Power".
  OPTIONAL { <http://example.org/patient/123> foaf:familyName "Simpson". }
  OPTIONAL { <http://example.org/patient/123> schema:birthDate ?birthdate. }
  <http://example.org/patient/123> schema:weight ?g_118.
  ?g_118 schema:value "125"^^<http://www.w3.org/2001/XMLSchema#decimal>.
  OPTIONAL {
    <http://example.org/patient/123> schema:memberOf ?medoffice.
    ?medoffice schema:address ?g_119.
    ?g_119 schema:streetAddress ?medoffice_street;
      schema:postalCode ?medoffice_zip.
  }
}
```
